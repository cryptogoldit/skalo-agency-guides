# Skalo Agency — AI & Digital Automation Guides

Benvenuto nel repository pubblico di **Skalo.agency**.
Qui condividiamo guide strategiche, architetture logiche e risorse open-source su intelligenza artificiale, automazione dei processi per PMI, strategie social e ottimizzazione di campagne ADV.

- `/guide/`: Guide strategiche in formato Markdown, complete di diagrammi Mermaid.js.
- `/casi-studio/`: Focus tecnico sui nostri 12 progetti reali.
- `/risorse/`: Script e configurazioni utili pronte all'uso.
